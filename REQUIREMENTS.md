# ТЗ: Сайт-приглашение на свадьбу Артёма и Яны

## 1. Общее описание

Одностраничный (single-page) сайт-приглашение на свадьбу. Гости переходят по ссылке,
видят красиво оформленное приглашение и могут подтвердить своё участие через форму RSVP.
Ответы сохраняются в БД и дублируются в Telegram жениху.

Проект публикуется на GitHub → подходит для портфолио.

---

## 2. Стек технологий

| Слой | Технология |
|------|-----------|
| Backend | Python 3.11+, FastAPI |
| База данных | SQLite (через SQLAlchemy или aiosqlite) |
| Шаблоны | Jinja2 (FastAPI встроенная поддержка) |
| Стили | CSS (чистый, без фреймворков) + Google Fonts |
| Уведомления | Telegram Bot API (httpx / aiogram) |
| Хостинг | Любой VPS или Railway / Render (указать в README) |

---

## 3. Дизайн и стиль

- **Стиль:** романтический, цветочный, нежный
- **Цветовая палитра:** пастельные тона — пудровый розовый `#F4C2C2`, кремовый `#FFF8F0`,
  sage-зелёный `#B2C9AD`, белый `#FFFFFF`
- **Шрифты:**
  - Заголовки: `Cormorant Garamond` (Google Fonts) — элегантный курсив
  - Текст: `Lato` или `Raleway` — чистый, читаемый
- **Декоративные элементы:** цветочные SVG-разделители между секциями,
  тонкие рамки, полупрозрачные оверлеи
- **Анимации:** плавное появление секций при скролле (Intersection Observer),
  пульсирующее сердечко в hero-блоке

## 4. Структура страницы (секции)

### 4.1 Hero — Заглавный экран

- Имена крупно: **Артём & Яна**
- Дата свадьбы: **19 сентября 2026**
- Короткий подзаголовок: *«Мы приглашаем вас разделить с нами этот особенный день»*
- Кнопка-якорь «Подтвердить участие» → прокрутка к форме RSVP
- Декоративные цветочные иллюстрации по краям

### 4.2 Таймер обратного отсчёта

- Live-счётчик: **дней / часов / минут / секунд** до 19.09.2026 12:00
- Подпись: *«До нашей свадьбы осталось»*
- Обновляется каждую секунду через JavaScript

### 4.3 История знакомства *(опционально — текст заполняется вручную)*

- Короткий романтический текст о паре (placeholder в шаблоне)
- Фото пары (опционально: `<img>` с переменной в шаблоне, если фото не нужно — секция скрыта)

### 4.4 Программа дня

| Время | Событие | Место |
|-------|---------|-------|
| 11:30 | Сбор гостей | ЗАГС Петроградского района |
| 12:00 | Церемония бракосочетания | Большая Монетная ул., 17-19, СПб |
| ~13:00 | Фотосессия у ЗАГСа | Там же |

- Иконки для каждого пункта (emoji или SVG)

### 4.5 Место проведения

- Название: **ЗАГС Петроградского района**
- Адрес: **Большая Монетная ул., 17-19, Санкт-Петербург**
- Встраиваемая карта: Яндекс.Карты или Google Maps iframe
- Кнопка «Построить маршрут» → открывает Яндекс.Карты/Google Maps

### 4.6 Дресс-код *(секция может быть раскрыта/скрыта флагом в конфиге)*

- Заголовок: *«Дресс-код»*
- Цветовые круги с рекомендованной палитрой (пастельные тона)
- Короткий текст-пожелание (placeholder)

### 4.7 RSVP — Форма подтверждения

Поля формы:
- **Имя** (text, required)
- **Буду / Не смогу прийти** (radio или select, required)
- **Пищевые предпочтения / аллергии** (textarea, optional)
- **Комментарий** (textarea, optional)

После отправки:
- Данные сохраняются в SQLite (таблица `rsvp`)
- Telegram-бот отправляет сообщение хозяину с данными гостя
- Гость видит страницу/сообщение благодарности: *«Спасибо! Ждём вас!»*
- Повторная отправка с тем же именем → обновление записи (upsert)

## 5. Backend — FastAPI

### Маршруты

| Метод | URL | Описание |
|-------|-----|---------|
| `GET` | `/` | Отдаёт главную страницу (Jinja2-шаблон) |
| `POST` | `/rsvp` | Принимает форму, сохраняет в БД, отправляет в TG |
| `GET` | `/admin` | Простая страница со списком всех RSVP (защита паролем) |

### Модель БД (SQLite, таблица `rsvp`)

```sql
CREATE TABLE rsvp (
    id        INTEGER PRIMARY KEY AUTOINCREMENT,
    name      TEXT NOT NULL,
    attending TEXT NOT NULL,  -- 'yes' / 'no'
    food_note TEXT,
    comment   TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 6. Telegram-уведомления

- Используется `python-telegram-bot` или чистый `httpx` (sendMessage API)
- Переменные окружения: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`
- Формат сообщения в Telegram:

```
🎊 Новый ответ на приглашение!

👤 Имя: Иван Иванов
✅ Придёт: Да
🍽 Пожелания: нет аллергий
💬 Комментарий: Очень рады за вас!

📅 19.09.2026 12:00 — RSVP получен
```

## 9. Нефункциональные требования

- **Адаптивность:** сайт должен корректно отображаться на мобильных (≥320px),
  планшетах и десктопах (CSS Grid / Flexbox)
- **Производительность:** загрузка страницы < 3 сек на мобильном 3G
- **Безопасность:** `SECRET_ADMIN_PASSWORD` хранится в `.env`, не в коде;
  admin-маршрут защищён HTTP Basic Auth
- **Язык:** только русский
- **Браузеры:** Chrome, Safari, Firefox (последние 2 версии)

## 10. Деплой (рекомендации для README)
